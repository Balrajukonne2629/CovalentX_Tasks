# CovalentX_Tasks

## 📌 Overview
This repository contains a collection of tasks completed as part of the **CovalentX program**.  
Each task demonstrates practical skills in **HTML**, **CSS**, and **JavaScript**, focusing on building a strong foundation in front-end web development.

---

## 🗂️ Repository Structure
The repository is organized into task folders:

- **task-1/** — Basic setup and introductory exercise  
- **task-2/** — HTML structure and semantic elements  
- **task-3/** — CSS styling and layout improvements  
- **task-4/** — JavaScript interactivity and DOM manipulation  
- **task-5/** — Integrated project combining HTML, CSS, and JS  

---

## 🛠️ Tech Stack
- **HTML (33.9%)** — Semantic structure and content  
- **CSS (41.3%)** — Styling, layout, and responsive design  
- **JavaScript (24.8%)** — Dynamic behavior and logic
